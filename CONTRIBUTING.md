# Contributing to TexAPI

Thank you for your interest in contributing to TexAPI!

## How to Contribute

### Report Issues
- Found a bug? [Open an issue](https://github.com/texapi/texapi/issues)
- Include your model, endpoint, and error message

### Suggest Features
- Have an idea? [Start a discussion](https://github.com/texapi/texapi/discussions)

### Improve Documentation
- Fix typos, add examples, improve clarity
- Submit a PR with your changes

### Share Examples
- Add code examples in new languages
- Show integrations with popular frameworks

## Code of Conduct

Be respectful, constructive, and inclusive. We're all here to build better developer tools.

## Questions?

- Discord: [discord.gg/texapi](https://discord.gg/texapi)
- Email: support@texapi.dev
